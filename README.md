## 🚀 Getting Started

### Prerequisites

- Flutter SDK
- Android Studio / VS Code
- Android Emulator or Physical Device

### Installation

```bash
git clone https://github.com/RaghuvanshAgarwal/todoey-flutter.git
```

```bash
cd todoey
```

Install packages

```bash
flutter pub get
```

Run the application

```bash
flutter run
```
